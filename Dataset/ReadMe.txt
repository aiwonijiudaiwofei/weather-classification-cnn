We do not own the copyright of the images contained in this dataset, they are used under different Creative Commons licenses that are properly referred to in files located in the "Licensing" folder.
The "Licensing" folder contains several CSV files that present relevant information
for license information about the images contained in the dataset.
There is one CSV file per each category, relating the image file name to its author and the name of the license it is referring to.
Also, the folder contains different text files with the conditions and information relevant to each type of license.
This dataset is intended for non-commercial and research purposes.
